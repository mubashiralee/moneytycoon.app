Introduction

Welcome to Money Tycoon ("we," "our," or "us"). This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our mobile application Money Tycoon (the "App"). Please read this privacy policy carefully. If you do not agree with the terms of this privacy policy, please do not access the App.

What changed in v3.0.0+: Earlier versions of Money Tycoon stored all game data locally on your device. Each of these is consent-gated, and Money Tycoon remains fully playable offline with all premium content unlocked through in-app purchases — cloud sync is opt-in convenience, not a requirement.

Cloud Save and Account Sync (added v3.0.0)

Starting with v3.0.0, Money Tycoon offers optional cross-device cloud save sync, powered by Firebase. The following Firebase services are used:

Firebase Auth (anonymous)

To support cross-device sync without requiring an email address, Money Tycoon creates an anonymous Firebase Auth account on first launch. This account holds no personally identifying information — it is a randomly generated UID. You may optionally link this account to a Google sign-in (Sign in with Google) or email/password if you wish to access your save from a different install. Unlinking a Google account is supported in the in-app Settings → Account screen. See Firebase Privacy and Security.

Firebase Firestore (cross-device save mirror)

When cloud save is enabled, the App mirrors 14 system subcollections of your game state into Firebase Firestore under a document keyed to your Firebase Auth UID. The mirrored subcollections include:

  • Player metadata (display name, country, current age)
  • Cash, bank, and credit card balances
  • Owned businesses, properties, and trading positions
  • Stock holdings and portfolio history
  • Achievement progress
  • Daily Hub state, season pass tier, prestige history
  • Tutorial / onboarding progress
  • Notification preferences and FCM token
  • Persona / lifestyle metrics
  • Trading and banking ledgers

Cloud save is governed by Firestore security rules that enforce per-player isolation — only the device authenticated as your Firebase Auth UID can read or write your data. Sync is bandwidth-budgeted (≤50 KB per device per day under normal play) and is paused when the device is on cellular and "Wi-Fi only" is selected in Settings.

You can disable cloud save at any time in Settings → Account → Cloud Sync. Disabling unlinks the local install from the cloud copy but does not delete the cloud copy. To delete the cloud copy, use Settings → Account → Delete cloud data (described under "Data Deletion" below).

Firebase Cloud Messaging (added v2.5.0)

If you opt in to push notifications, the App registers a Firebase Cloud Messaging (FCM) registration token issued by Google. This token allows our backend to deliver streak reminders, leaderboard updates, daily challenge notifications, and major release announcements — each opt-in is granular and per-category in Settings → Notifications. The FCM token is not personally identifying on its own and is rotated periodically by the operating system. You can disable any push category at any time in the App's Settings, or revoke notification permission system-wide via OS settings.

Per-user frequency cap: ≤ 7 push notifications per week, server-enforced.

Firebase Analytics, Crashlytics, and Remote Config (added v2.10.2 / consent-gated v3.1.1)

To improve the App and diagnose stability issues, Money Tycoon uses three additional Firebase services. All three are off by default and only enabled when you grant explicit consent (TCF v2 / Google Consent Mode v2 signals):

  • Firebase Analytics: Aggregate, anonymous gameplay metrics (screen views, feature usage, session length). No personal data, no purchase amounts, no account identifiers. Used to identify features that need improvement.
  • Firebase Crashlytics: Crash stack traces, device model, OS version, and app version. The custom user ID set on Crashlytics is your Firebase Auth UID (anonymous) — never your email, name, or device IMEI. Used to triage stability bugs.
  • Firebase Remote Config: Server-side feature flags and balance-tuning values (e.g., event rewards). Reads only — no data is sent from your device to Remote Config.

If you decline consent on the first-launch dialog or via Settings → Manage Ad Preferences, all three services remain disabled, and your device sends no analytics or crash data to Google. You can revisit and reverse your consent decision at any time.

Google Play Games Services (added v2.5.0)

If you sign in to Google Play Games (optional), we receive your Google Play Games Player ID and display name to render you on the global leaderboards. Money Tycoon submits the following non-personal game-state values to Google Play Games servers:

  • Number of properties owned
  • Monthly passive income (USD-converted)
  • Number of prestige resets
  • Number of achievements unlocked
  • Largest single-trade profit (USD-converted)

You can opt out by not signing in to Google Play Games — the game remains fully playable with anonymous (local-only) leaderboards. Google's data handling for Play Games is documented in Google's Privacy Policy.

Tycoon Card Image Sharing (added v2.5.0)

The "Share Tycoon Card" feature renders a 1080×1920 PNG image on your device (containing your in-game stats — net worth, business count, property count, achievement count, prestige count) and lets you share it via Android's standard share sheet (iMessage relay, WhatsApp, Twitter, Instagram Stories, etc.). The image is generated on-device only — Money Tycoon does not upload, store, or transmit the rendered card. Once shared, the receiving app handles the image per its own privacy policy.

Advertising (added v2.9.0, expanded v2.13.3)

Money Tycoon displays optional rewarded video advertisements that you initiate yourself by tapping a "Watch Ad for Reward" button. Ads are never forced. Watching a rewarded ad delivers small in-game bonuses (e.g., speed boost, daily bonus claim, education accelerator). Mogul Premium subscribers receive an ad-free experience.

Unity Ads SDK (primary ad network, added v2.9.0)

Money Tycoon uses Unity Ads SDK 4.13.0 to serve rewarded video ads. Unity Ads may collect and process information including your device's advertising identifier (AAID), approximate location (based on IP address), and ad interaction data to serve relevant ads and measure their performance.

  • Unity Ads serves personalized or non-personalized ads depending on your consent preferences and applicable regional laws.
  • See Unity Game Player and App User Privacy Policy
  • See Unity Privacy Policy

Liftoff Monetize / Vungle (mediation partner, added v2.13.3)

For certain ad placements, Money Tycoon uses Liftoff Monetize (formerly Vungle) as a mediation partner alongside Unity Ads. Liftoff may collect device identifiers, IP-based approximate location, and ad interaction events to serve and measure ads. The same consent state you provided for Unity Ads applies to Liftoff.

  • See Liftoff Privacy Policy

Google User Messaging Platform (UMP, added v2.9.0)

For users in the EEA, UK, Switzerland, and California, we use Google's User Messaging Platform (UMP) SDK to present a consent form on first launch. UMP collects your consent state per IAB TCF v2 / Google Consent Mode v2 signals and stores it in SharedPreferences on your device. The consent state is forwarded to Unity Ads, Liftoff, and our Firebase services. You can revisit and change your consent at any time via Settings → Manage Ad Preferences.

Important: Until you have completed the UMP consent flow (or are outside a region where consent is required), Money Tycoon does not initialize Firebase Analytics or Crashlytics, does not serve personalized ads, and does not set Google's advertising cookies.

Optional Pro Trading Data Feeds (added v2.6+)

Money Tycoon's Pro Trading mode uses real-world stock and crypto market data to drive simulated trades. These feeds are read-only — Money Tycoon sends no data to them about you. The App requests price quotes anonymously via HTTPS:

  • Alpha Vantage — Daily/weekly equity quotes. Privacy Policy
  • Finnhub — Real-time and historical equity quotes. Privacy Policy
  • Alpaca — Real-time market data via WebSocket. Privacy Notice

These feeds are activated only when you open the Pro Trading screen. If you have your own API keys (Alpha Vantage / Finnhub / Alpaca) and enter them in Settings → Pro Trading → API Keys, those keys are stored locally on your device only and used only for outgoing requests to the respective provider.

Data Storage and Security

Local Storage

The bulk of game progress is stored locally on your device using Android's SharedPreferences, Room database, and DataStore. This local data is:

  • Not transmitted to our servers (except via the cloud-sync system above when you opt in)
  • Not shared with third parties for marketing
  • Deleted when you uninstall the App or clear App data

Cloud Storage (Firestore)

When you opt in to cloud save, your game state mirror is stored in Google Firebase Firestore (us-central1 region). Firestore encrypts data at rest and in transit. Access is governed by Firestore security rules that enforce per-player isolation.

Security Measures

We implement appropriate technical and organizational security measures to protect your information, including:

  • HTTPS for all network traffic; cleartext traffic explicitly disabled
  • Firebase Auth UID per-row authorization on Firestore
  • Local data encryption at rest (Android's default protection)
  • Regular security assessments
  • R8/ProGuard obfuscation in release builds

Your Rights and Choices

Data Deletion

You have multiple ways to delete your Money Tycoon data:

  • Uninstall the App — deletes all local data on your device. If you have cloud save enabled, the cloud copy is not affected by uninstall.
  • Clear App Data — wipes all local data via your device's Settings → Apps → Money Tycoon → Storage → Clear data.
  • Reset Game (in-app) — Settings → Account → Reset Game. Wipes all local data including SharedPreferences (CharitableGivingRepository, LeaderboardInstrumentation, MogulPassport, DailyHub, LiveEvents, FlashSale, season manager, prefs cache) and removes the cloud copy from Firestore.
  • Delete Cloud Data only — Settings → Account → Delete cloud data. Removes your Firestore document while leaving local data intact. Useful if you want to keep playing offline.
  • Email request — write to capitalcraftgames@gmail.com from the email associated with your linked Google sign-in (if any) and we will process the deletion within 30 days.

Opt-Out of Analytics and Personalized Ads

You have several options to limit data collection:

  • UMP consent revoke: Settings → Manage Ad Preferences re-launches the consent form. Selecting "Reject all" disables Firebase Analytics, Crashlytics, and turns off personalized ads.
  • Personalized Ads: On Android, go to Settings → Google → Ads and toggle "Opt out of Ads Personalization", or use the "Delete advertising ID" option.
  • Subscription: Upgrading to Mogul Premium removes all advertisements from the App entirely.
  • Cloud Sync: Settings → Account → Cloud Sync toggles cloud save on/off.
  • Notifications: Each notification category (streak reminders, daily challenges, leaderboard updates, releases) has a separate Settings toggle. OS-level notification permission overrides all in-app toggles.
